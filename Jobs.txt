we have 2 types of jobs in linux 
 => at jobs : single time jobs or non repetative jobs the jobs will automatically erase once executed
 => cron jobs :cron excutes jobs repetaed , this jobs not removed untill we remove manually
 atjobs=>

            cd /etc/at.deny => to deny particular user 
			cd /etc/sysconfig/at
			cd /var/log/cron = to check  execution status 
commands : at now +2 min
           atq => to see job list
		   at -l => to see h=joblist
		   atrm => to remove job with job id
